# macEventButton
For a hardware button to kick off macEvent

use WS2912B for led

Planed steps.

Power on

set status pin to high

flash led green

Connect to wifi

error flash red

send mqtt message

error flash amber

ok flash led green 

set status pin to low

The external power will turn off.
